Facções

Shinigami

Guerreiros espirituais responsáveis por purificar Hollows e manter o equilíbrio entre os mundos.

Quincy

Humanos com poder espiritual capazes de derrotar Hollows usando energia espiritual condensada.

Hollows / Arrancars

Espíritos corrompidos que representam ameaça constante, incluindo guerreiros poderosos como Espadas e Arrancars.

Sobre o projeto

Este é um projeto inicial de estudo com HTML e CSS, criado para treinar estrutura, estilo, responsividade e organização de páginas temáticas.

A ideia é evoluir esse projeto por versões, adicionando mais personagens, facções, páginas internas e, no futuro, interatividade com JavaScript.
